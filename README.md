## VaneBot

To build (see documentation for aidoboy/discordpp for dependencies):

    git clone --recursive https://github.com/nagakawa/vane.git
    cmake .
    make

### Functionality

`Please help me.` to see list of commands

`x/eks "S{mp@ tEkst/` to convert to IPA
